# Sample Research Papers

Download 10-15 PDFs from arXiv.org and place them here.

## Recommended Topics:
1. **Machine Learning** (3-4 papers)
   - Search: "random forest", "XGBoost", "ensemble learning"
   - https://arxiv.org/search/?query=random+forest&searchtype=all

2. **Computer Vision** (3-4 papers)
   - Search: "object detection", "vision transformers", "image classification"
   - https://arxiv.org/search/?query=object+detection&searchtype=all

3. **Natural Language Processing** (3-4 papers)
   - Search: "transformers", "BERT", "language models"
   - https://arxiv.org/search/?query=BERT+transformers&searchtype=all

4. **Survey Papers** (2-3 papers)
   - Search: "deep learning survey", "neural networks survey"
   - https://arxiv.org/search/?query=deep+learning+survey&searchtype=all

## Quick Links:
- https://arxiv.org/ - Main site
- https://arxiv.org/list/cs.LG/recent - Latest ML papers
- https://arxiv.org/list/cs.CV/recent - Latest CV papers
- https://arxiv.org/list/cs.CL/recent - Latest NLP papers

## Instructions:
1. Click on any paper title
2. Click "Download PDF" button
3. Save to this folder (test_papers/)
4. Rename to something simple if needed (e.g., paper1.pdf, paper2.pdf)

Once you have 10-15 papers, run:
```bash
python test_pdf_reader.py
```
